Weather Data API

URL format for single temperature: http://127.0.0.1:5000/api/v1/station/YYYY-MM-DD

Example: http://127.0.0.1:5000/api/v1/10/1988-10-25

URL format for temperatures for all dates: http://127.0.0.1:5000/api/v1/station

Example: http://127.0.0.1:5000/api/v1/10

URL format for temperatures for a specific year: http://127.0.0.1:5000/api/v1/annual/station/YYYY

Example: http://127.0.0.1:5000/api/v1/annual/10/1988
